# 开发档案

## 基本目标
开发一个电子商城网站

## 最小可运行需求

- 注册(添加用户)，登录(验证)
- 添加商品信息，主要信息有title, price, description, photo, quantity
- 列出商品
- 添加商品到购买车
- 结账支付

## 任务流程

- 搭建MERN架构
- 部署项目的自动化流程
- 编写Product Module
- 编写User Module
- 编写Order Module
- 连接主要Module
- 部署和发布项目 v1.0
